📱 使い方ガイド

🚀 基本の使い方

1. レシート撮影
📷撮影ボタンをタップしてレシートを撮影

2. 内容確認
AIが自動で金額・店名・科目を読み取ります

3. 編集（必要に応じて）
カードをタップして内容を修正

4. 申請送信
💸まとめて申請ボタンで送信

📸 撮影のコツ

  • 明るい場所で撮影する
  • レシート全体を画面に収める
  • 平らな場所に置いて真上から撮影
  • 手ぶれしないようにゆっくりと

📏 長いレシートの撮影

1枚に収まらない長いレシートの場合：

  1. まず上半分を撮影
  2. 「前の明細の続き」にチェック✓
  3. 下半分を撮影
  4. 金額が自動で合算されます

※画像は両方とも保存され、カードに「×2」と表示されます

✏️ 編集機能

レシートカードをタップすると編集画面が開きます：

  • 金額：読み取りが不正確な場合に修正
  • 店名：正しい店舗名に変更
  • 科目：適切な経費科目を選択

🔧 よくある問題

金額が0円になる
→ 明るい場所で再撮影するか、手動で入力

店名が読めない
→ 編集画面で正しい店名を入力

科目がわからない
→ 管理者に確認してください

📋 申請前チェック

  • ✅ 金額が正しく入力されている
  • ✅ 店名が正しく入力されている
  • ✅ 適切な科目が選択されている
  • ✅ レシート画像がはっきり見える

📖 管理者ガイド

⚡ 承認・却下

申請一覧：未処理の経費申請が表示されます

  • 金額修正：必要に応じて明細の金額や科目を修正可能
  • 承認：内容に問題なければ承認
  • 却下：不適切な申請を却下（申請者に通知）
  • 差し戻し：修正が必要な場合、理由を入力して差し戻し

📁 科目管理

経費の分類に使う科目マスタを管理します

  • 追加：科目名・税率・キーワードを入力して追加
  • 編集：既存の科目をタップして修正
  • 削除：不要な科目を削除

※キーワードはOCR自動判定に使用されます

📊 月次レポート

月別の収支状況を確認できます

  • 収入：入金登録した金額の合計
  • 支出：承認済み経費の合計
  • 残高：繰越＋収入−支出
  • 拠点フィルタ：拠点別に絞り込み可能

📅 日別集計

日別・担当者別の経費集計を確認できます

  • 月を選択して集計期間を指定
  • 担当者ごとの経費総額を一覧表示

💸 支払い・履歴

承認済み申請の支払い状況を管理します

  • 支払済にする：経費を精算したらボタンをタップ
  • 履歴確認：過去の申請内容を確認・再判定

※入金・調整系（DEP-）は支払い対象外のためボタン非表示

💰 入金・調整登録

現金入金や調整を登録します

  • 入金：小口現金の補充など
  • 出金：経費申請外の出金
  • 調整：過不足の調整

※登録した内容は月次レポートの収支に反映されます
